# General JavaScript Coding Drills

A collection of small, practical coding challenges and utility functions often asked in interviews.

- **[algorithm-problems.js](algorithm-problems.js)**: Assorted algorithm implementations.
- **[array-flat.js](array-flat.js)**: Implementing `Array.prototype.flat`.
- **[array-uniqueValue.js](array-uniqueValue.js)**: Finding unique values in arrays.
- **[coercion-problem.js](coercion-problem.js)**: Tricky type coercion examples.
- **[get-by-path.js](get-by-path.js)**: Accessing nested object properties by string path (e.g., `get(obj, 'a.b.c')`).
- **[longest-string.js](longest-string.js)**: Finding the longest string in an array.
- **[move-element.js](move-element.js)**: Moving array elements.
- **[number-pairs.js](number-pairs.js)**: Finding pairs with a target sum.
- **[scope-problem.js](scope-problem.js)**: Scope-related challenges.
- **[shared-number.js](shared-number.js)**: Finding shared numbers between arrays.
- **[sleep.js](sleep.js)**: Implementing a sleep function.
- **[small-questions.js](small-questions.js)**: Quickfire JS questions.
- **[smart-multiplication.js](smart-multiplication.js)**: Curried multiplication.
- **[tricky-questions.js](tricky-questions.js)**: Common "gotcha" questions.
